## 📖 简介

mindGit 是一个 Chrome 浏览器扩展，帮助你追踪和可视化网页之间的跳转关系。当你从网页 A 点击链接到网页 B，再到网页 C 时，mindGit 会以树形结构记录这个路径，让你轻松回溯浏览的脉络。

## ✨ 功能特点

### 🌲 树形结构可视化
- 清晰的树状图展示浏览路径
- 父子节点用连接线相连
- 支持展开/折叠子节点
- 不同层级有不同的视觉样式

### 📊 多会话管理
- 支持创建多个独立的浏览会话
- 区分不同的浏览任务（如工作、学习、娱乐）
- 自动命名会话，也可自定义名称

### 🌙 暗色模式
- 一键切换亮色/暗色主题
- 主题偏好自动保存
- 暗色模式配色护眼舒适

### 🔍 快速搜索
- 弹窗式搜索，快速定位会话和节点
- 支持按标题、URL 搜索
- 搜索结果高亮定位，一键跳转

### 📤 数据导出
- 支持导出为 Markdown 格式（笔记软件友好）
- 支持导出为 HTML 格式（可用浏览器打开）
- 方便存档和分享浏览记录

### ⚡ 智能记录
- 自动追踪页面跳转，无需手动操作
- 智能识别重复访问，自动合并节点
- 前进/后退操作正确处理

### 🎨 优雅的界面
- 现代化的卡片式设计
- 平滑的过渡动画
- 清晰的视觉层次

## 📦 安装指南

### 方法一：Chrome 应用商店（推荐）

> ⚠️ 当前版本尚未发布到 Chrome 应用商店，请先使用开发者模式安装。

### 方法二：开发者模式安装（本地安装）

#### 步骤 1：下载源码

```bash
# 使用 git 克隆
git clone https://github.com/RumiaLunnar/mindGit.git

# 或者下载 ZIP 文件
# 访问 https://github.com/RumiaLunnar/mindGit/archive/refs/heads/master.zip
```

#### 步骤 2：打开 Chrome 扩展管理页面

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 或者在菜单中选择：更多工具 → 扩展程序

#### 步骤 3：开启开发者模式

在扩展管理页面右上角，找到并开启「**开发者模式**」开关：

```
┌─────────────────────────────────────────┐
│ 开发者模式 [开关按钮]                    │
└─────────────────────────────────────────┘
```

#### 步骤 4：加载扩展

1. 点击左上角的「**加载已解压的扩展程序**」按钮
2. 在弹出的文件选择器中，选择你下载的 `mindGit` 文件夹
3. 确保选择的是**包含 `manifest.json` 的文件夹**，而不是子文件夹

#### 步骤 5：验证安装

安装成功后：
- 页面会显示 `mindGit` 扩展卡片
- 浏览器工具栏会出现 🌳 图标
- 点击图标即可开始使用

### 方法三：拖拽安装（CRX 文件）

如果你已经下载了 `.crx` 文件：

1. 打开 `chrome://extensions/`
2. 开启开发者模式
3. 将 `.crx` 文件拖拽到扩展管理页面
4. 点击「添加扩展程序」确认

## 🚀 使用方法

### 开始记录

1. 点击工具栏的 🌳 图标打开 mindGit
2. 点击 ➕ 创建新会话（或自动使用当前会话）
3. 正常浏览网页，mindGit 会自动记录你的跳转路径
4. 随时点击图标查看当前的浏览树

### 界面说明

```
┌─────────────────────────────────────┐
│  🌳 mindGit    🌙 🔄 ➕ 🔍 📤 ⚙️   │  ← 头部：主题、刷新、新建、搜索、导出、设置
├─────────────────────────────────────┤
│  [选择会话...]        🗑️            │  ← 会话选择器
├─────────────────────────────────────┤
│  📊 浏览会话 · 2个起点 · 5个页面    │  ← 统计信息
├─────────────────────────────────────┤
│  │                                  │
│  ├── 🌐 知乎首页                    │  ← 根节点（起点）
│  │   │                              │
│  │   ├── 📄 文章 A                  │  ← 子节点
│  │   │   └── 📄 相关推荐            │  ← 更深层级
│  │   │                              │
│  │   └── 📄 文章 B                  │
│  │                                  │
│  └── 🌐 GitHub                      │  ← 另一个起点
│      └── 📄 项目主页                │
├─────────────────────────────────────┤
│  [清空所有]  [全部展开] [全部折叠]  │  ← 底部操作栏
└─────────────────────────────────────┘
```

### 快捷键

| 操作 | 说明 |
|------|------|
| 点击节点 | 在新标签页打开链接 |
| 点击 ▼/▶ | 展开/折叠子节点 |
| 点击 🌙/☀️ | 切换暗色/亮色主题 |
| 点击 🔍 | 打开搜索弹窗 |
| 点击 📤 | 导出当前会话 |
| ESC | 关闭弹窗/取消搜索 |
| ↑↓ | 搜索结果导航 |

## 💡 使用场景

### 📚 深度阅读与研究
当你在阅读一篇文章时，经常被文中的链接吸引，不断跳转到新的页面。几小时后，你想回到最初的那篇文章，却发现自己已经在几十个标签页中迷失了方向。

**mindGit 帮你**：
- 清晰记录从「首页 → 文章 A → 相关推荐 → 深度内容」的完整路径
- 随时回溯阅读脉络，快速找回起点
- 支持搜索跳转的正确关联（右键搜索也能记录来源）

### 🔧 技术问题排查
遇到 Bug 时，你在 Google、Stack Overflow、GitHub、官方文档之间反复跳转，最终解决了问题，但想整理解决方案时却记不清查阅了哪些资料。

**mindGit 帮你**：
```
Google 搜索 "React useEffect 依赖问题"
└── Stack Overflow 相关问题
    └── GitHub Issue #1234
        └── 相关 PR 修复
            └── React 官方文档
```
- 完整保存技术调研的路径
- 方便后续整理成技术博客或文档

### 🛒 购物决策对比
想买一款电子产品，你在京东、淘宝、亚马逊、品牌官网之间反复对比价格、参数、评价，最后却忘了在哪个页面看到最优价格。

**mindGit 帮你**：
- 记录所有比价页面的跳转路径
- 每个商品页面都有来源追溯
- 快速回到最优惠的购买页面

### 🎓 知识学习探索
在 Wikipedia 或专业学习网站上，从一个概念链接到另一个概念，不知不觉已经打开了十几个页面，想系统整理学习笔记时却理不清知识脉络。

**mindGit 帮你**：
```
人工智能
└── 机器学习
    ├── 监督学习
    │   └── 决策树
    └── 神经网络
        ├── 卷积神经网络
        └── 循环神经网络
```
- 将碎片化学习路径可视化
- 形成完整的知识图谱
- 方便复习和系统整理

### 📊 竞品分析调研
产品经理需要分析多个竞品的功能，在产品官网、帮助文档、用户评价页面之间跳转，需要整理分析思路。

**mindGit 帮你**：
- 区分不同竞品的浏览路径
- 支持多会话管理（每个竞品一个会话）
- 快速定位和对比关键页面

### 🔍 信息收集整理
做 PPT、写报告时需要收集大量参考资料，从搜索引擎到各类资讯网站，最后整理引用来源时费时费力。

**mindGit 帮你**：
- 自动记录所有参考页面
- 一键打开原始页面获取引用信息
- 按浏览路径整理资料逻辑

### ✨ 更多可能性
- **旅游攻略**：从游记网站 → 景点介绍 → 交通信息 → 酒店预订
- **菜谱学习**：从美食博客 → 食材选购 → 烹饪技巧 → 视频教程
- **股票研究**：从财经新闻 → 公司财报 → 行业分析 → 交易平台
- **游戏攻略**：从 Wiki → 论坛讨论 → 视频攻略 → 道具数据库

## 🔒 隐私说明

- **本地存储**：所有数据存储在浏览器本地，不会上传到任何服务器
- **权限最小化**：仅请求必要的权限（标签页、存储、导航）
- **数据安全**：不会收集个人隐私信息

## 🛠️ 技术栈

- JavaScript (ES6+)
- CSS3 (Flexbox、CSS Variables)
- Chrome Storage API
- Chrome Tabs API
- Chrome WebNavigation API

## 🔧 开发指南

### 本地开发

1. 克隆仓库
   ```bash
   git clone https://github.com/RumiaLunnar/mindGit.git
   cd mindGit
   ```

2. 在 Chrome 中加载扩展（开发者模式）

3. 修改代码后，在扩展管理页面点击刷新按钮重新加载

```bash

# 或者手动打包
```
