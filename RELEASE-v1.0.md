## 🎉 MindGit v1.0

### ✨ 初始功能

- **🌲 树形结构可视化** - 清晰的树状图展示浏览路径，父子节点关系一目了然
- **📊 多会话管理** - 支持创建多个独立的浏览会话，区分不同的浏览任务
- **🌙 暗色模式** - 一键切换亮色/暗色主题，主题偏好自动保存
- **⚡ 智能记录** - 自动追踪页面跳转，无需手动操作
- **🎨 优雅界面** - 现代化的卡片式设计，平滑的过渡动画

### 🔧 核心能力

- 自动记录网页跳转关系
- 支持展开/折叠子节点
- 不同层级有不同的视觉样式
- 智能识别重复访问
- 前进/后退操作正确处理

### 📦 安装

下载 `mindgit-v1.0.tar.gz` 并解压，在 Chrome 扩展开发者模式下加载。
